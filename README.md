# Projetos Acadêmicos - Faculdade

Este repositório centraliza os códigos, projetos práticos e listas de exercícios desenvolvidos por mim durante o curso no **Instituto Federal de Sergipe (IFS), Campus Lagarto**.

O objetivo é registrar minha evolução e documentar a aplicação prática dos conceitos teóricos aprendidos em aula.

## 📂 Organização do Repositório

* **`POO-TYPESCRIPT/`**: Sistema de modelagem acadêmica e gestão de campus desenvolvido para consolidação de conceitos fundamentais de Programação Orientada a Objetos com TypeScript.
